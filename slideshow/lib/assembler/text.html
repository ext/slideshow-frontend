Using this option you can create new slides with text only. The text will be applied to the predefined background if specified in the configuration.

${Markup(item.render(content.get(item.name, '')))}

Text is parsed using Pango Markup. Allowed html-tags is <b>, <big>, <i>, <s>, <sub>, <sup>, <small>, <span>, <tt> and <u>.
Variables placed in curly brackets {} can be sub­stituted by POSTing to /slides/update/${id and id or ':id'}.
